EchoJournal is a web application that enables users to create and manage digital journals with image attachments, along with public and private visibility options. Users can seamlessly perform CRUD operations on their journals and export them as PDF documents for offline access. All journals are securely stored within user profiles, authenticated through JWT and Google OAuth.

The platform provides insightful activity analytics through interactive graph, allowing users to track their journaling patterns over time. Other users can like or dislike public journals, triggering notifications for journal owners. Additionally, EchoJournal features a community discussion space that allows users to engage in meaningful conversations.

### Technologies Used: 

React, Express, Node.js, MongoDB, React Redux, Cloudinary, Google OAuth

### How to run

Run ```npm install``` inside both folders

Frontend: ```npm run dev```

Backend: ```nodemon index.js```

Make sure to create appropriate ```.env``` files in both frontend and backend folders
